# Gym Connect

Description:  
Gym Connect is a comprehensive database system designed for LSU University Recreation (UREC). It efficiently manages member information, membership types, class schedules, instructors, attendance records, and facility locations, streamlining operations and enhancing member services.

Features:
- Member Management: Maintain detailed records of members, including their membership categories (e.g., current students, alumni, parents).
- Class Scheduling: Organize and manage class types, schedules, and instructor assignments.
- Attendance Tracking: Monitor member attendance to various classes and events.
- Facility Management: Keep track of different locations within UREC where classes and events are held.

Installation:
1. Clone the Repository:
   ```bash
   git clone https://github.com/yourusername/gym-connect.git
   cd gym-connect
   ```
2. Set Up the Database:
   - Ensure SQLite is installed on your system.
   - Configure the database connection in the `application.properties` file.
3. Run Migrations:
   - Use the appropriate command to set up the database schema.

Usage:
- Adding Members: Input new member data and assign the appropriate membership type.
- Scheduling Classes: Create and modify class schedules, assigning instructors and locations.
- Recording Attendance: Log attendance for each class to monitor member participation.
- Reporting: Generate reports to analyze member engagement and class popularity.
